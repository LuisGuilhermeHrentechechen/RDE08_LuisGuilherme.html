Table

Éla assume por função desenvolver tabelas, sendo composta por ( table , tr , th e por fim td ). Onde vai começar com o table, logo depois vai vim o tr que define as linhas da tabela, th é como se fosse o título da tabela, e td é o conteúdo que vai ser preenchido dentro da tabela.

Del

A tag del serve para riscar o texto que foi escrito. Como por exemplo: IFPR.

Ins

Já a tag ins serve para sublinhar uma palavra ou frase do texto, como: IFPR.

Sub

Sub serve para deixar uma palavra ou letra pequena, mas para a parte de baixo da linha, exemplo: 1928.

Sup

Muito parecido com o sub o sup tem praticamente a mesma função, mas a diferença é que com o sup a letra fica pequena na parte de cima da linha, como se ela fosse elevada, assim: 1928.

blockquote

Blockquote é usado quando se precisa fazer uma grande citações, é possivel ver a ultilização dele na wikipédia, exemplo:

Timothy John Berners-Lee é um físico e cientista da computação, de 62 anos, nascido em Londres, capital do Reino Unido. É reconhecido como a figura central no processo de desenvolvimento da internet por ser responsável pela primeira comunicação bem sucedida entre cliente e um servidor via HTTP, em 1990.

Q

Com o mesmo intuito do blockquote a tag q é usada para fazer citações curtas, como por exemplo: Essa atividade leva tempo.

Abbr

Tem por sua finalidade a abreviação de frases, o Abbr é usado com esse fim. exemplo: IFPR

Address

A tag address é basicamente usada quando existe a necessidade de colocar dados de uma pessoa, ou empresa.

Cite

Agora por vez quando se precisa de uma tag para se dirigir ao nome de uma obra por exemplo usa-se a tag cite, como por exemplo: Tom e jerry o Filme.

Pre

Essa tag pre ela consegue manter espaçoes entre o texto escrito e também as quebras de linhas sem precisar do br .

Kbd

É usada para "demostrar" uma função do teclado, kbd é uzada para isso, exemplo o Esc que é usado para sair de uma página
